Project Description
This project is a responsive HTML and external CSS personal portfolio website. It's based on the Week 2-3 bio and contact pages, with some CSS styling added, a Flexbox navigation bar and some media queries for mobile responsiveness.

 Pages Included
This includes personal bio, profile image, and the hobbies list in the file named index.html
- `contact.html` - contains the contact form and other contact information.
- `style.css` - All external styles for the website are included in this file.

Features
- Responsive portfolio layout
- External CSS styling
- Flexbox navigation bar
- Integrating the contact page with the rest of the site.
- Media query for screens that are less than 600 pixels wide
- Change of background color on smaller screens.
- When viewing on a smaller screen, the profile section will be displayed side by side.
- On smaller screens the profile section will appear vertically stacked.
- Hobbies list
- Properly labeled contact form
- Clear description of the image on the alt attribute text box.
- Semantic HTML elements: header, nav, main, section, article, footer.
